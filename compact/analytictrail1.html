Analytic trail 1 (Private vs state ownership in selected countries)

Research question

Does the largest countries (based on GDP) sorted by continent have similar attitude to private vs state ownership of business? For example is there an attitude towards private ownership in these countries?

Select wave

In the first step you choose which wave (time period) you are interesting in, in this case the period '2010-2014'.

Select countries

You now select the countries of interest.

Select question

The survey questions could be viewed as a list (left) or as a tree structure (right), and has the ability to search.

Responses

Here responses of the survey question are showed for the selected time period (wave). It can be viewed in a spreadsheet or with different charts (as seen below).

Maps

This view let the user choose an indicator and view the result on a world map (left image). The map can be viewed with different indicators and filter based on indicator value.

Time line

Displays multiple waves and helps with discovering trends. It can be viewed as a list (left image), or as charts (right image).

Insights

No similar attitude between the countries seemed to exist based on fluctuating attitude in different time periods. There doesn't seem to be a particulary high preference for private or state ownership of business, instead it seems to fluctuate between different time periods. The tool could be improved by giving an overview of the answer distribution. And when using the time line you had to scroll up and down a lot when comparing countries which was difficult.
